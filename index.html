슬라이드 배너, 메뉴 등을 구현할 수 있으며, bxslider 등을 이용한 슬라이드 제작도 하였습니다.

참여도 )

기획 100% / 디자인 100% / 코딩 100%

기존 쇼핑몰 특징 )

많은 상품만 아주 작게 나열되어 있었고, 사용한 컬러가 거의 없었음.

리뉴얼 )

pc - 1920px 기준으로 제작.
화면에 꽉 차는 메인배너와 걸리시한 악세서리 쇼핑몰에 걸맞는 컬러를 활용해
리뉴얼하였음. 상품의 이미지를 키워 상세페이지에 들어가지 않아도 상품이
눈에 띄도록 하여 편의성을 높였음.
메인페이지,카테고리페이지,상세페이지 총 3개를 제작하였고, 쇼핑몰에 어울리는
배너 또한 제작하였음.

서브페이지 부분에서 작은 화면에 글씨와 배경이 겹쳐져 수정하고 싶은 부분이 있었음.
